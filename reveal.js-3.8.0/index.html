SVG

Scalable Vector Graphics

What is SVG?

SVG is a vector graphic format-based on XML and is used to display a variety of graphics on the Web and other environments.

Basic Shapes

Rectangle

x / y - The x/y positions of the top left corner of the rectangle

rx / ry - The x/y radius of the corners of the rectangle


              < svg width="400" height="200">
              < rect x="120" y="20" rx="20" ry="20" width="150" height="150"
              style="fill:red; stroke:white; stroke-width:5;"/> < /svg>

Circle

r - The radius of the circle

cx / cy - The x/y positions of the center of the circle


              < svg width="400" height="200"> 
              < circle cx="185" cy="100" r="80" 
              stroke="white" stroke-width="3" fill="red"/>
              < /svg>

Ellipse

rx / ry - The x/y radiuses of the ellipse

cx / cy - The x/y positions of the center of the ellipse


              < svg width="400" height="200">
              < ellipse cx="200" cy="100" rx="150" ry="80" 
              stroke="white" stroke-width="3" fill="red"/>
              < /svg>

Line

x1 / y1 - The x/y positions of point 1

x2 / y2 - The x/y positions of point 2


              < svg width="400" height="200">
              < line x1="15" y1="25" x2="350" y2="180" 
              stroke="red" stroke-width="7"/>
              < /svg>

Polyline

points - A list, where each point must contain two numbers, an x coordinate and a y coordinate


              < svg width="400" height="200">
              < polyline points="20,40 60,45 80,60 100,140 180,160 320,120"
               style="fill:none; stroke: red; stroke-width:7"/>
              < /svg>

Polygon

points - A list, where each point must contain two numbers, an x coordinate and a y coordinate


              < svg width="400" height="200">
                < polygon points="160,10 120,198 320,78 60,78 320,198"
                style="fill:red; stroke:white; stroke-width:5;"/>
              < /svg>

Path

d - A list of commands and parameters used by those commands


              < svg width="400" height="200">
                < path d="M200 10 L130 180 L275 180 Z" 
                fill="red" stroke="white" stroke-width="5"/>
              < /svg>

How to add svg to the page ?

Svg insertion methods:

- Img
- Background-image
- Inline Svg
- Object

Img or background-image


              < img src="icon.svg" alt="Just a simple icon">

              .icon {
              background-image: url('icon.svg');
              height: 64px;
              width: 64px;
              }

Advantages:

  • Simple use
  • Caching

Disadvantages:

  • No possible to interact with graphics

Object


              < object type="image/svg+xml" data="your.svg" id="object"
              class="icon"> < /object>

Advantages:

  • Can be manipulated with CSS
  • Scripts, animation work( if they are described inside SVG)

Disadvantages:

  • No caching

Inline Svg


              < svg height="100" width="500" class="icon">
                < ellipse cx="240" cy="50" rx="220" ry="30" fill="red" />
                < ellipse cx="240" cy="50" rx="190" ry="20" fill="white"/>
              < /svg>

Advantages:

  • The full potential of SVG is available
  • No loading of external files

Disadvantages:

  • No caching
  • Page code gets dirty
